# Mansoury - Furniture & Home Appliances Website

## Overview
Mansoury is a responsive e-commerce landing page for furniture and home appliances.  
The project was created using HTML and Tailwind CSS and supports mobile, tablet, and desktop layouts.

---

## Technologies Used

- HTML5
- Tailwind CSS
- Font Awesome

---

## Features

- Responsive Design
- Product Showcase
- Promotional Sections
- Modern UI Design
- Scrollable Product Cards
- Discount Banners
- Footer Navigation

---

## Responsive Design

The website adapts to different screen sizes:

- Mobile
- Tablet
- DesktoP
